Hoja de Ruta Estratégica

De MVP interno → a Producto vendible · v1.0

◆ El Enfoque
No competimos contra ClickUp. Usamos ClickUp gratis para gestionar el día a día mientras construimos el MVP como base arquitectónica del software de facturación — el verdadero producto vendible. Cada línea de código del MVP (auth, API, entidades, Blazor) se reutiliza en el producto final para PyMEs.
Track A — Gestión (ClickUp)
Herramienta de trabajo diaria. Tareas, horas, colaboración. Se configura una vez y se usa siempre.
Track B — Desarrollo (Código)
Construcción del MVP como fundación técnica del producto de facturación futuro.
